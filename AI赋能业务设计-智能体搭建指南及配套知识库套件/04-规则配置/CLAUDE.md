## 🎯 使用场景

### 场景1: 个人规则配置
```
1. 在Trae IDE中创建个人规则
2. 复制 通用配置/个人规则.md 的内容
3. 应用到AI助手的个人规则设置中
4. AI助手将按照这些规则执行任务
```

### 场景2: 项目规则配置
```
1. 在Trae IDE中创建项目规则
2. 复制 项目规则/6A执行规则.md 的内容
3. 应用到项目的规则设置中
4. 在项目中输入"6A"开头的内容激活6A工作流
```

### 场景3: 规则组合使用
```
1. 个人规则: 全局通用规则（始终生效）
2. 项目规则: 6A执行规则（项目特定规则）
3. 两者结合: 个人规则作为基础，项目规则作为增强
```
