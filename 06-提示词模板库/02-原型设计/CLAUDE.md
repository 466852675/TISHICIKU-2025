## 📋 模块概述

**模块名称**: 原型设计专家  
**模块类型**: 配置库  
**模块定位**: 专业原型设计专家配置，涵盖PC端、APP端设计和UI规范分析

### 核心功能
- 🎨 PC端界面设计（看板风格、系统风格、简约风格）

- 🔍 UI规范分析和逆向工程
- 💻 生成可直接运行的HTML代码

## 📖 关键文件说明

### PC端设计专家

#### 1. 看板风格专家.md
- **功能**: 数据可视化大屏设计
- **适用场景**: 数据可视化看板、大屏展示
- **技术栈**: HTML + Tailwind CSS + Chart.js
- **特点**: 响应式设计、暗色主题、多种图表类型

#### 2. 系统风格专家.md
- **功能**: 企业管理系统界面设计
- **适用场景**: 企业级管理系统、后台管理界面
- **技术栈**: HTML + Tailwind CSS + FontAwesome
- **特点**: 标准化的企业系统风格、功能完整

#### 3. 简约风格专家.md
- **功能**: 现代简洁设计风格
- **适用场景**: 现代简洁界面、轻量级应用
- **技术栈**: HTML + Tailwind CSS
- **特点**: 简洁现代、交互友好

### APP端设计专家

#### 1. 国网风专家.md
- **功能**: 国家电网标准风格APP设计
- **适用场景**: 电力行业APP、国网标准应用
- **技术栈**: HTML + Tailwind CSS + FontAwesome
- **特点**: 主色#009D85、适配iPhone 15 Pro、符合国网标准

#### 2. 弥散风专家.md
- **功能**: 现代弥散风格APP设计
- **适用场景**: 现代弥散渐变效果、时尚应用
- **技术栈**: HTML + Tailwind CSS
- **特点**: 弥散渐变、现代设计、视觉冲击力强

#### 3. 苹果风专家.md
- **功能**: iOS Human Interface Guidelines风格
- **适用场景**: iOS标准设计、苹果风格应用
- **技术栈**: HTML + Tailwind CSS
- **特点**: 符合iOS设计规范、原生体验

### UI规范专家

#### UI规范专家.md
- **功能**: UI设计逆向工程，像素级界面解构
- **适用场景**: 界面分析和复现、AI生图提示词生成
- **复杂度**: ⭐⭐⭐⭐⭐
- **特点**: 
  - 六大模块分析（布局、色彩、字体、图标、交互、细节）
  - 专业AI生图提示词生成
  - 像素级界面解构

## 🎯 使用场景

### 典型工作流程

#### PC端设计流程
```
业务需求
    ↓
选择风格（看板/系统/简约）
    ↓
生成HTML代码
    ↓
在浏览器中运行测试
    ↓
功能测试专家验证
```

#### APP端设计流程
```
APP需求
    ↓
选择风格（国网风/弥散风/苹果风）
    ↓
生成HTML代码
    ↓
在移动端浏览器中测试
    ↓
UI规范分析（如需要）
    ↓
功能测试专家验证
```

#### UI分析流程
```
上传UI截图
    ↓
UI规范专家分析
    ↓
生成六大模块分析
    ↓
生成AI生图提示词
    ↓
用于界面复现或设计参考
```
