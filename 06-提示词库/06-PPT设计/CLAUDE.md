# 06-PPT设计

## 模块定位

本模块是专业的PPT设计专家库，提供多种高端商务风格的PPT设计专家配置。适用于企业汇报、项目展示、产品发布等场景。

## 各IDE使用方式

### Trae IDE（推荐）

**方式1：创建专用智能体**

1. 在Trae中点击"创建智能体"
2. 命名智能体（如"墨蓝PPT设计专家"）
3. 从本模块复制对应专家的.md内容到系统提示词
4. 保存并使用

**方式2：项目规则配置**

在 `.trae/rules/project_rules.md` 中添加：
```markdown
## PPT设计规范
- 使用科技蓝图风格
- 主色调：墨蓝 #0F172A
- 字体：思源黑体
- 比例：16:9
```

### Cursor

**方式1：.cursorrules配置**

在项目根目录创建 `.cursorrules`：
```markdown
# PPT设计专家规则
你是PPT设计专家，擅长科技蓝图风格...
[粘贴专家提示词内容]
```

**方式2：对话中使用**

在Chat中先发送：
```
你现在是"墨蓝PPT设计专家"，请按照以下规则设计PPT：
[粘贴专家提示词内容]
```

### 其他IDE

在AI对话中先发送专家提示词定义角色，再发送具体PPT设计需求。

## 使用示例

### 示例1：科技公司产品发布会PPT

**选择专家**：【墨蓝】现代科技蓝图风专家

**输入需求**：
```
设计一个AI产品发布会的PPT，包含：
1. 封面：产品名称"智能助手Pro"
2. 产品概述：3个核心功能
3. 技术架构：系统架构图
4. 应用场景：4个行业案例
5. 未来规划：发展路线图
6. 结束页：联系方式

要求：
- 每页都要有视觉冲击力
- 数据用图表展示
- 保持整体风格一致
```

### 示例2：工业项目汇报PPT

**选择专家**：【午夜蓝】数字孪生蓝图风专家

**输入需求**：
```
设计一个智能制造项目汇报PPT，包含：
1. 项目背景
2. 数字孪生架构
3. 实施效果数据
4. 成本收益分析
5. 下一步计划
```

## 设计规范速查

### 色彩方案

| 风格 | 背景色 | 主色 | 强调色 |
|------|--------|------|--------|
| 墨蓝科技 | #0F172A | #FFFFFF | #2DD4BF |
| 工业蓝 | #1E3A8A | #F8FAFC | #60A5FA |
| 炭黑精密 | #1F2937 | #E5E7EB | #9CA3AF |
| 午夜蓝 | #0F172A | #F1F5F9 | #38BDF8 |
| 冷灰白 | #F3F4F6 | #1F2937 | #4B5563 |

### 字体规范

- **标题**：思源黑体 Bold / 微软雅黑 Bold
- **正文**：思源黑体 Regular / 微软雅黑
- **数据**：JetBrains Mono / Consolas

### 布局原则

1. **16:9比例**：所有设计保持16:9宽屏比例
2. **留白原则**：保持30%以上留白
3. **对齐原则**：使用网格系统对齐元素
4. **层次原则**：标题>副标题>正文>注释
