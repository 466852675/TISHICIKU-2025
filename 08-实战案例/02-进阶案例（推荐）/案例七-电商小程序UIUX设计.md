# 案例七：电商小程序UI/UX设计

> 🟡 **进阶案例（推荐）** | **预计耗时**：30分钟  
> **使用专家**：7个设计专家（看板/系统/简约/国网风/弥散风/苹果风/UI规范）  
> **配套Skill**：ui-ux-pro-max, frontend-design, canvas-design  
> **输出成果**：完整的小程序UI设计稿（7种风格）+ 设计规范文档

---

## 📋 案例概述

### 案例背景

为一个电商小程序设计完整的UI/UX，包括首页、商品详情、购物车、订单、个人中心等核心页面。

### 学习目标

- 掌握7个设计专家的不同风格
- 学会使用ui-ux-pro-max Skill优化设计
- 了解设计规范文档的编写

---

## 🚀 多IDE操作指南

### Trae用户

#### 步骤1：创建设计专家智能体

创建多个智能体，分别使用不同风格：

1. **简约风格专家**：`06-提示词库/02-原型设计/PC端/简约风格专家.md`
2. **弥散风专家**：`06-提示词库/02-原型设计/APP端/弥散风专家.md`
3. **苹果风专家**：`06-提示词库/02-原型设计/APP端/苹果风专家.md`
4. **UI规范专家**：`06-提示词库/02-原型设计/UI规范专家.md`

#### 步骤2：设计首页（多风格对比）

使用不同专家设计同一页面：

**简约风格版本**：
```
请设计电商小程序首页（简约风格）：

功能模块：
1. 顶部搜索栏
2. 轮播Banner（3张）
3. 分类入口（8个图标）
4. 限时秒杀区
5. 推荐商品列表（双列瀑布流）
6. 底部导航栏（首页/分类/购物车/我的）

设计要求：
- 极简主义风格
- 大量留白
- 黑白灰为主，点缀品牌色
- 字体：苹方/PingFang
- 圆角设计
```

**弥散风格版本**：
```
请设计电商小程序首页（弥散风格）：

功能模块同上

设计要求：
- 弥散渐变背景
- 玻璃拟态效果
- 柔和阴影
- 流体形状装饰
- 配色：粉紫渐变、蓝绿渐变
```

#### 步骤3：UI规范逆向工程

使用UI规范专家：
```
请分析以下设计稿，生成UI规范文档：

输入：
- 首页设计稿（简约风格）
- 商品详情页设计稿
- 购物车页面设计稿

输出UI规范：
1. 色彩系统（主色/辅助色/中性色/功能色）
2. 字体规范（字号/字重/行高）
3. 间距系统（8px基准）
4. 圆角规范
5. 阴影规范
6. 图标规范
7. 组件规范（按钮/输入框/卡片/列表）
8. 布局栅格
```

#### 步骤4：使用Skill优化设计

```
@ui-ux-pro-max 请优化刚才的小程序设计：

优化要求：
1. 应用"现代极简"设计风格
2. 使用"科技蓝"配色方案（#3B82F6）
3. 字体搭配：标题用"Inter"，正文用"Noto Sans SC"
4. 添加微交互动效
5. 确保无障碍设计（WCAG 2.1 AA标准）
```

#### 步骤5：生成前端代码

```
@frontend-design 请将UI设计稿转换为小程序代码：

技术要求：
1. 使用微信小程序原生语法
2. 使用Flex布局
3. 组件化开发
4. 添加注释说明
5. 输出：pages/home/index.wxml + index.wxss
```

#### 步骤6：设计宣传物料

```
@canvas-design 请设计小程序推广海报：

海报尺寸：1080x1920px（手机竖屏）

内容：
- 标题：XX商城小程序上线啦！
- 副标题：新用户首单立减50元
- 二维码（小程序码位置）
- 商品展示（3-4个热销商品）

风格：简约现代，突出促销信息
```

## 7个设计专家对比

| 专家 | 风格特点 | 适用场景 |
|------|---------|---------|
| 看板风格专家 | 数据可视化、深色主题 | 后台管理系统 |
| 系统风格专家 | 企业级、专业严谨 | 企业管理系统 |
| 简约风格专家 | 极简、留白、现代 | C端产品 |
| 国网风专家 | 电力行业、绿色主题 | 电力行业APP |
| 弥散风专家 | 渐变、玻璃拟态 | 时尚、年轻化产品 |
| 苹果风专家 | iOS规范、精致 | iOS应用 |
| UI规范专家 | 逆向工程、规范输出 | 设计系统建设 |
