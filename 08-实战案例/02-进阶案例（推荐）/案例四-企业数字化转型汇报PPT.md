# 案例四：企业数字化转型汇报PPT

> 🟡 **进阶案例（推荐）** | **预计耗时**：20分钟  
> **使用专家**：9个PPT设计专家（墨蓝/工业蓝/炭黑/午夜蓝/海蓝/深邃工业蓝/冷灰白/浅米灰/象牙白）  
> **配套Skill**：pptx, theme-factory  
> **输出成果**：3套不同风格的PPT（每套15-20页）

---

## 📋 案例概述

### 案例背景

为某制造企业制作数字化转型项目汇报PPT，向董事会汇报数字化建设成果和未来规划。

### 学习目标

- 掌握9个PPT设计专家的不同风格
- 学会使用pptx Skill生成PPT文件
- 了解theme-factory的主题应用能力

---

## 🚀 多IDE操作指南

### Trae用户

#### 步骤1：创建PPT专家智能体

创建3个智能体，分别使用不同风格的专家：

**智能体1：墨蓝科技风**
- 名称：墨蓝PPT专家
- 提示词：`06-提示词库/06-PPT设计/01-科技蓝图风格/【墨蓝】现代科技蓝图风专家.md`

**智能体2：工业蓝数字孪生风**
- 名称：工业蓝PPT专家
- 提示词：`06-提示词库/06-PPT设计/02-工业数字孪生风格/【工业蓝】数字蓝图与工业感风专家.md`

**智能体3：冷灰白手稿风**
- 名称：冷灰白PPT专家
- 提示词：`06-提示词库/06-PPT设计/03-手绘艺术风格/【冷灰白】精益蓝图与战略手稿风专家.md`

#### 步骤2：输入汇报内容

```
请基于以下内容制作数字化转型汇报PPT：

汇报主题：XX制造集团数字化转型三年成果汇报

汇报大纲：
1. 封面：XX制造集团数字化转型成果汇报（2024年度）
2. 目录：项目概述、建设成果、效益分析、未来规划
3. 项目概述：
   - 背景：传统制造业面临效率瓶颈
   - 目标：3年实现全流程数字化
   - 投资：总投资5000万元
4. 建设成果：
   - 智能工厂：建设5G+工业互联网平台
   - 数据中台：整合12个业务系统数据
   - AI应用：质检准确率提升至99.5%
   - 数字孪生：实现生产线虚拟仿真
5. 效益分析：
   - 生产效率提升35%
   - 运营成本降低20%
   - 产品不良率下降40%
   - 交付周期缩短30%
6. 未来规划：
   - 2025年：全面智能化
   - 2026年：产业链协同
   - 2027年：生态化运营
7. 结束页：感谢聆听

要求：
- 每页PPT有明确的标题和要点
- 使用图表展示数据和对比
- 添加适当的图标和插图
- 保持风格统一
```

#### 步骤3：生成PPT文件

```
@pptx 请将刚才设计的PPT转换为.pptx格式，要求：
1. 使用标准16:9比例
2. 每页布局清晰，层次分明
3. 图表使用Excel数据源
4. 添加过渡动画效果
5. 输出文件：数字化转型汇报-墨蓝版.pptx
```

#### 步骤4：应用主题

```
@theme-factory 请为刚才的PPT应用"科技蓝"主题，调整：
1. 主色调：#0F172A（墨蓝）
2. 强调色：#3B82F6（科技蓝）
3. 字体：标题使用思源黑体，正文使用微软雅黑
4. 添加渐变背景和阴影效果
```
