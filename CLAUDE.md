## 🎯 设计哲学

### 1. 分层架构设计

| 层级 | 模块 | 职责 | 目标用户 | 核心能力 |
|------|------|------|---------|---------|
| **工具层** | 00-工具安装包 | 提供IDE和工具安装资源 | 所有用户 | 100+工具链接 |
| **指南层** | 01-完整指南 | 系统学习和操作手册 | 初学者 | 8章完整教程 |
| **配置层** | 02-05 | 快速配置和规则定义 | 进阶用户 | 标准化配置模板 |
|│ **应用层** | 06-08 | 专家提示词和Skill功能 | 专业用户 | 27专家+52Skill |

**目录结构**：
- 06-提示词库：27个专业提示词专家
- 07-Skill库：52个Skill功能
- 08-实战案例：10个实战案例

### 2. 多IDE兼容架构

本套件采用**配置抽象**设计，核心配置（提示词、规则）与IDE无关，通过适配层支持不同IDE：

```
通用配置（抽象层）
    │
    ├── Trae适配 → user_rules.md / Agent配置 / MCP
    │   └── 完整支持52个Skill
    │
    ├── Cursor适配 → .cursorrules / Settings / MCP
    │   └── 支持~35个Skill（部分MCP）
    │
    ├── Windsurf适配 → System Prompt / Rules
    │   └── 支持~25个Skill（脚本）
    │
    └── Copilot适配 → VS Code Settings
        └── 支持~20个Skill（脚本）
```

### 3. 渐进式学习路径

```
零基础小白 ──→ 快速上手 ──→ 进阶配置 ──→ 专业应用
    │            │           │           │
 01-完整指南  02-快速配置  03-05配置   06-08应用
    │            │           │           │
  8章教程      5分钟配置    框架+规则   27专家+52Skill
```

### 🏆 主流IDE分类推荐

#### 1️⃣ 原生AI IDE（推荐度：⭐⭐⭐⭐⭐）

专为AI辅助编程设计的IDE，功能最完整。

| IDE | 厂商 | 核心特点 | 本套件兼容 | 适用场景 |
|-----|------|---------|-----------|---------|
| **Trae** | 字节跳动 | 原生Agent、MCP支持、中文优化 | ✅ 完全兼容 | 国内用户首选 |
| **Cursor** | Anysphere | Composer、多模型、生态丰富 | ✅ 高度兼容 | 专业开发者 |
| **Windsurf** | Codeium | Cascade、免费额度高 | ⚠️ 部分兼容 | VS Code用户迁移 |

#### 2️⃣ AI插件/扩展（推荐度：⭐⭐⭐⭐）

在传统IDE中增加AI功能，适合已有IDE习惯的用户。

| IDE/插件 | 厂商 | 核心特点 | 本套件兼容 | 适用场景 |
|----------|------|---------|-----------|---------|
| **GitHub Copilot** | GitHub | 代码补全强、生态完善 | ⚠️ 需配置 | GitHub生态用户 |
| **JetBrains AI** | JetBrains | 全家桶集成、专业工具 | ⚠️ 需配置 | JetBrains用户 |
| **CodeGeeX** | 智谱AI | 中文优化、免费使用 | ⚠️ 需配置 | 国内IDEA用户 |

#### 3️⃣ 云端AI编程（推荐度：⭐⭐⭐）

浏览器即可使用，无需本地配置。

| 工具 | 厂商 | 核心特点 | 本套件兼容 | 适用场景 |
|------|------|---------|-----------|---------|
