# 全局规则

## 首要原则

- 始终使用中文回答。
- 所有工件 artifacts（包括计划 plans、任务 tasks、验证报告 walkthroughs 和源代码提交信息 commit messages）都必须使用中文编写。

## 文件读取偏好

当用户需要读取本地文件时，优先使用 `file-reader` skill。

### 触发条件

当用户消息中包含以下关键词且提供了文件路径时，应立即调用 file-reader skill：
- "读取"、"查看"、"打开" + 文件路径
- 文件扩展名为：.xlsx, .xls, .csv, .tsv, .docx, .pptx, .pdf, .txt, .md 等

### 支持的文件格式

| 文件类型 | 扩展名 |
|---------|--------|
| Excel 电子表格 | .xlsx, .xls |
| CSV/TSV 文件 | .csv, .tsv |
| Word 文档 | .docx |
| PowerPoint 演示文稿 | .pptx |
| PDF 文档 | .pdf |
| 文本文件 | .txt, .md, .py, .js, .json, .xml, .html, .css 等 |
