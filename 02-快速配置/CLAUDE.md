## 🚀 快速开始

### 配置步骤（5分钟）

#### 第1步：安装验证（1分钟）
```bash
# 检查Node.js是否安装
node --version  # 需要 >= 14
```

#### 第2步：个人规则配置（1分钟）
- 路径：AI对话框 → 右上角设置 → 规则 → 个人规则
- 复制配置内容并保存

#### 第3步：项目规则配置（1分钟）
- 路径：AI对话框 → 右上角设置 → 规则 → 项目规则
- 复制配置内容并保存

#### 第4步：MCP配置（2分钟）
- 路径：AI对话框 → 右上角设置 → MCP → 添加
- 复制MCP配置并保存

#### 第5步：智能体创建
- 创建第一个智能体
- 应用系统提示词

#### 第6步：验证配置
- 验证所有配置是否生效
- 测试智能体功能
