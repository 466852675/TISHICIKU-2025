# 工具安装网址

> 📦 **开发工具安装链接集合** - 快速访问所有必需工具的官方下载页面

---

## 🚀 必需工具

### Trae CN IDE

**官方网址**: https://www.trae.cn/

**说明**: AI辅助开发IDE，本套件的核心工具

**安装要求**:
- 支持 Windows、macOS、Linux
- 建议使用最新版本

---

### Node.js

**官方网址**: https://nodejs.cn/download/

**说明**: JavaScript 运行环境，用于运行 MCP 服务和构建工具

**版本要求**: >= 14（推荐使用 LTS 版本）

**安装验证**:
```bash
node --version
```

---

### Python

**官方网址**: https://pythonlang.cn/downloads/

**说明**: Python 运行环境，部分工具和脚本需要

**版本要求**: >= 3.8（推荐使用最新稳定版）

**安装验证**:
```bash
python --version
```

---

## 📋 快速安装清单

| 工具 | 网址 | 必需性 | 版本要求 |
|------|------|--------|----------|
| Trae CN IDE | https://www.trae.cn/ | ✅ 必需 | 最新版 |
| Node.js | https://nodejs.cn/download/ | ✅ 必需 | >= 14 (LTS) |
| Python | https://pythonlang.cn/downloads/ | ⚠️ 可选 | >= 3.8 |

---

## 🔧 安装顺序建议

1. **Trae CN IDE** - 核心开发工具
2. **Node.js** - 运行 MCP 服务必需
3. **Python** - 按需安装（部分功能需要）

---

## ✅ 安装后验证

安装完成后，请运行以下命令验证：

```bash
# 验证 Node.js
node --version

# 验证 Python（如已安装）
python --version
```

验证通过后，可以继续配置 Trae IDE 环境，参考：[快速配置清单](../02-快速配置/快速配置清单.md)
