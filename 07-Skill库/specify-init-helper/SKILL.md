# Specify 项目初始化

当用户提到以下场景时自动执行：

- 开发流程
- 交付流程
- 规范流程
- 初始化 Specify
- 开始新项目

## 执行操作

1. 打开 PowerShell 终端
2. 输入并执行 `specify-cn init .`
3. 等待用户确认后续操作
