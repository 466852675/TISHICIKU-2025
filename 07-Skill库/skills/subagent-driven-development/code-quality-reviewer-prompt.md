# 代码质量审查提示模板

在调度代码质量审查子代理时使用此模板。

**目的：** 验证实施构建良好（干净、测试良好、可维护）

**仅在规范合规审查通过后调度。**

```
Task tool (superpowers:code-reviewer):
  使用requesting-code-review/code-reviewer.md中的模板

  实施内容：[来自实施者的报告]
  计划或要求：[计划文件]中的任务N
  BASE_SHA：[任务之前的提交]
  HEAD_SHA：[当前提交]
  描述：[任务摘要]
```

**代码审查返回：** 优势、问题（严重/重要/次要）、评估
