# 技能设计的说服原则

## 概述

LLM 与人类对相同的说服原则做出反应。理解这种心理学有助于你设计更有效的技能——不是为了操纵，而是为了确保即使在压力下也能遵循关键实践。

**研究基础：** Meincke 等人（2025）用 N=28,000 次 AI 对话测试了 7 种说服原则。说服技术使遵守率提高了一倍以上（33% → 72%，p < .001）。

## 七大原则

### 1. 权威
**是什么：** 对专业知识、证书或官方来源的服从。

**在技能中如何运作：**
- 命令式语言："你必须"、"绝不"、"总是"
- 不可协商的框架："无例外"
- 消除决策疲劳和合理化

**何时使用：**
- 强制执行纪律的技能（TDD、验证要求）
- 安全关键实践
- 已确立的最佳实践

**示例：**
```markdown
✅ 在测试之前写代码？删除它。重新开始。无例外。
❌ 在可行时考虑先写测试。
```

### 2. 承诺
**是什么：** 与先前行动、声明或公开宣言的一致性。

**在技能中如何运作：**
- 要求宣布："宣布技能使用"
- 强制明确选择："选择 A、B 或 C"
- 使用跟踪：用于清单的 TodoWrite

**何时使用：**
- 确保技能实际被遵循
- 多步骤流程
- 问责机制

**示例：**
```markdown
✅ 当你找到技能时，你必须宣布："我正在使用 [技能名称]"
❌ 考虑让你的伙伴知道你在使用哪个技能。
```

### 3. 稀缺性
**是什么：** 来自时间限制或有限可用性的紧迫感。

**在技能中如何运作：**
- 有时间限制的要求："在继续之前"
- 顺序依赖："在 X 之后立即"
- 防止拖延

**何时使用：**
- 立即验证要求
- 时间敏感的工作流
- 防止"我稍后再做"

**示例：**
```markdown
✅ 完成任务后，立即请求代码审查，然后再继续。
❌ 你可以在方便时审查代码。
```

### 4. 社会认同
**是什么：** 顺从他人所做或被认为正常的事情。

**在技能中如何运作：**
- 普遍模式："每次"、"总是"
- 失败模式："没有 Y 的 X = 失败"
- 确立规范

**何时使用：**
- 记录普遍实践
- 警告常见失败
- 强化标准

**示例：**
```markdown
✅ 没有 TodoWrite 跟踪的清单 = 步骤被跳过。每次。
❌ 有些人发现 TodoWrite 对清单有帮助。
```

### 5. 统一性
**是什么：** 共享身份，"我们感"，群内归属感。

**在技能中如何运作：**
- 协作语言："我们的代码库"、"我们是同事"
- 共享目标："我们都想要质量"

**何时使用：**
- 协作工作流
- 建立团队文化
- 非等级实践

**示例：**
```markdown
✅ 我们是同事，一起工作。我需要你诚实的技术判断。
❌ 如果我错了，你应该告诉我。
```

### 6. 互惠性
**是什么：** 回报所收到的好处的义务。

**如何运作：**
- 谨慎使用 - 可能感觉操纵性
- 技能中很少需要

**何时避免：**
- 几乎总是（其他原则更有效）

### 7. 喜好
**是什么：** 偏好与我们喜欢的人合作。

**如何运作：**
- **不要用于合规**
- 与诚实反馈文化冲突
- 创造谄媚

**何时避免：**
- 始终用于纪律执行

## 按技能类型的原则组合

| 技能类型 | 使用 | 避免 |
|------------|-----|-------|
| 强制执行纪律 | 权威 + 承诺 + 社会认同 | 喜好、互惠性 |
| 指导/技术 | 适度权威 + 统一性 | 重权威 |
| 协作 | 统一性 + 承诺 | 权威、喜好 |
| 参考 | 仅清晰度 | 所有说服 |

## 为什么有效：心理学

**明线规则减少合理化：**
- "你必须"消除决策疲劳
- 绝对语言消除"这是例外吗？"问题
- 明确的反合理化对策关闭特定漏洞

**实施意图创造自动行为：**
- 清晰的触发器 + 必需的行动 = 自动执行
- "当 X 时，做 Y"比"通常做 Y"更有效
- 减少遵守的认知负荷

**LLM 是类人的：**
- 在人类文本上训练，包含这些模式
- 权威语言在训练数据中先于遵守
- 承诺序列（声明 → 行动）经常被建模
- 社会认同模式（每个人都做 X）确立规范

## 道德使用

**合法：**
- 确保遵循关键实践
- 创建有效文档
- 防止可预测的失败

**不合法：**
- 为个人利益操纵
- 创造虚假紧迫感
- 基于内疚的遵守

**测试：** 如果他们完全理解，这种技术会服务于用户的真正利益吗？

## 研究引用

**Cialdini, R. B. (2021).** *影响力：说服心理学（新版和扩展版）。* Harper Business。
- 七种说服原则
- 影响力研究的实证基础

**Meincke, L., Shapiro, D., Duckworth, A. L., Mollick, E., Mollick, L., & Cialdini, R. (2025).** Call Me A Jerk: Persuading AI to Comply with Objectionable Requests. University of Pennsylvania.
- 用 N=28,000 次 LLM 对话测试 7 种原则
- 说服技术使遵守率提高 33% → 72%
- 权威、承诺、稀缺性最有效
- 验证 LLM 行为的类人模型

## 快速参考

设计技能时，问：

1. **它是什么类型？**（纪律 vs 指导 vs 参考）
2. **我试图改变什么行为？**
3. **哪些原则适用？**（通常权威 + 承诺用于纪律）
4. **我是否结合了太多？**（不要使用全部七个）
5. **这是否道德？**（服务于用户的真正利益？）
