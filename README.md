## 🎯 项目概述

**AI 智能体配置模板**是一个标准化、模块化的AI开发环境配置库，提供从框架搭建到生产部署的全套解决方案。通过精心设计的配置模板和专家级提示词，帮助用户**5分钟快速搭建**高质量的AI助手开发环境。

### ✨ 核心价值

- ⚡ **快速启动**: 开箱即用，大幅缩短开发周期
- 🌐 **多平台支持**: 兼容Claude、ModelScope、Trae CN等主流AI平台
- 🧠 **专家级提示词**: 经过实战验证的16个专业提示词专家配置
- 🧩 **模块化设计**: 灵活组合，按需配置
- 🏢 **企业级规范**: 遵循6A原则和行业最佳实践

### 🎯 目标用户

1. **个人开发者**: 快速搭建个人项目开发环境
2. **企业团队**: 标准化团队AI开发流程
3. **业务分析师**: 使用专业提示词专家完成业务设计任务
4. **零基础用户**: 通过完整指南快速上手

## ⚡ 5分钟快速开始

### 🎯 前置要求

- ✅ 已安装Node.js (版本 >= 14)
- ✅ 有AI平台账号 (Claude/ModelScope/Trae CN等)
- ✅ 基础的开发环境配置经验

### 🚀 快速配置步骤

#### 1. 选择平台配置

根据您使用的AI平台，选择对应的MCP服务配置：

```bash
# 复制MCP配置文件到对应位置
# 配置文件位置：AI赋能业务设计-智能体搭建指南及配套知识库套件/05-服务配置/MCP服务/通用mcp配置.json
```

#### 2. 安装MCP服务依赖

```bash
# 安装核心MCP服务
npm install -g @modelcontextprotocol/server-sequential-thinking
npm install -g @upstash/context7-mcp
npm install -g chrome-devtools-mcp
npm install -g @playwright/mcp
```

#### 3. 配置个人规则

在AI平台中创建个人规则，复制以下内容：

```markdown
# 个人规则
- 所有回复使用中文
- 保持一致的编码风格
- 为代码添加中文注释
- 复杂任务先拆分再执行
- 完成任务后说"写好了"
```

完整规则配置请参考：`AI赋能业务设计-智能体搭建指南及配套知识库套件/04-规则配置/通用配置/个人规则.md`

#### 4. 选择专家配置

根据任务需求，从 `06-提示词模板库/` 中选择对应的专家配置创建智能体。

**完成！** 🎉 您现在可以开始使用智能体配置模板了！

## 🛠️ 核心功能模块

### 📋 1. 框架搭建模块

专为React管理系统设计的系统级提示词配置，提供三个版本：

| 版本     | 文件                    | 适用场景       | 复杂度   |
| -------- | ----------------------- | -------------- | -------- |
| v1基础版 | `v1-system-prompt.md` | 简单管理系统   | ⭐⭐     |
| v2增强版 | `v2-system-prompt.md` | 中等复杂度系统 | ⭐⭐⭐   |
| v3专业版 | `v3-system-prompt.md` | 企业级系统     | ⭐⭐⭐⭐ |

**位置**: `AI赋能业务设计-智能体搭建指南及配套知识库套件/03-框架搭建/`

### ⚙️ 2. 服务配置模块

提供通用MCP服务配置，支持多种MCP服务：

- 🧠 **Sequential Thinking**: 顺序思考工具
- 📚 **context7**: 技术文档搜索服务
- 🌐 **Chrome DevTools MCP**: 浏览器调试工具
- 🤖 **playwright**: 浏览器自动化工具
- 📊 **图表服务**: mcp-server-chart、MCP-ECharts、quickchart

**位置**: `AI赋能业务设计-智能体搭建指南及配套知识库套件/05-服务配置/`

### 📏 3. 规则配置模块

提供AI执行规则和约束配置：

- **个人规则**: 全局通用规则配置
- **项目规则**: 6A执行规则（对齐→架构→原子化→审批→自动化→评估）

**位置**: `AI赋能业务设计-智能体搭建指南及配套知识库套件/04-规则配置/`

### 🧠 4. 提示词专家库

提供16个专业提示词专家配置，涵盖四大领域：

#### 📊 需求分析专家（3个）

- **可研生成专家**: 电网数字化项目可行性研究报告
- **PRD生成专家**: 完整产品需求文档创建

#### 🎨 原型设计专家（7个）

**PC端设计**:

- **看板风格专家**: 数据可视化看板设计
- **系统风格专家**: 企业级管理系统界面
- **简约风格专家**: 现代简洁设计风格

**APP端设计**:

- **国网风专家**: 国家电网标准风格
- **弥散风专家**: 现代弥散渐变效果
- **苹果风专家**: iOS Human Interface Guidelines风格

**UI规范专家**:

#### 🧪 测试验证专家（3个）

- **功能测试专家**: 页面功能全面测试
- **自动化测试专家**: 端到端自动化测试
- **文档质量校验专家**: 文档规范性检查

#### 🛠️ 通用专家（4个）

- **PPT生成专家**: 自动化演示文稿生成
- **周报生成专家**: 标准化工作周报
- **提示词生成专家**: AI提示词优化
- **网页生成专家**: 快速网页代码生成

**位置**: `AI赋能业务设计-智能体搭建指南及配套知识库套件/06-提示词模板库/`

## 🛡️ 安全与最佳实践

### 🔒 安全注意事项

- ❌ 不要在配置文件中硬编码API密钥
- ✅ 使用环境变量管理敏感信息
- ✅ 限制MCP服务的文件访问权限
- ✅ 定期更新API密钥和服务
